Initiates a new shapes's positional, colour and index data to be rendered with new shaders. Also takes user input parameters.

Function to initialise a new shape, vertex shader, fragment shader and shader program.

Parameters
isKitePasses in the boolean from main to determine if the CTRL key has been pressed to change the shape's values.
&toBeMovedLHas the A key been pressed in main. (Bool).
&toBeMovedRHas the D key been pressed in main. (Bool).
&toBeMovedUHas the W key been pressed in main. (Bool).
&toBeMovedDHas the S key been pressed in main. (Bool).

< Local Variable: numOfVerts - Used to set the size of the position and colourData arrays as a constant. (As is required to set an array size).

< Local Variable: numOfIndices - Used to set the size of the indicies array as a constant. (As is required to set an array size).

< Local Variable: revisedNumVerts - A pointer to the numOfVerts array size constant to make them temporarily dynamic ints to be changed depending on if the array will be storing the kite or cube data.

< Local Variable: revisedNumIndices - A pointer to the numOfIndices array size constant to make them temporarily dynamic ints to be changed depending on if the array will be storing the kite or cube data.

< Local Variable: positionData[] - The array to store the rendered shape's vertices' position data.

< Local Variable: colourData[] - The array to store the rendered shape's colour's data.

< Local Variable: indices[] - The array to store the rendered shape's indices' data.
